📌 Project Overview
MobileSphere is a Machine Learning based web application that predicts mobile phone prices using specifications like:
RAM
Storage
Battery Capacity
Camera
Brand

The project includes:
Data Cleaning
Exploratory Data Analysis (EDA)
Feature Encoding
Machine Learning Model Training
Model Comparison
Streamlit Deployment

🧠 Machine Learning Models Used
Linear Regression
Decision Tree Regressor
Random Forest Regressor
Gradient Boosting Regressor
XGBoost Regressor

🏆 Best Model: Random Forest
📊 Evaluation Metrics
R² Score (Accuracy)
MAE (Mean Absolute Error)
RMSE (Root Mean Square Error)
